Può assumere due ruoli principali:
1. **[[Variabile indicatrice]]**: indica se un elemento appartiene a un insieme $X \subseteq Y$
$$y= \begin{cases} 1 \qquad se \ x \in X \\ 0 \qquad se \ x \notin X\end{cases}$$
2. **[[Variabile logica]]**: indica il valore di verità di una *proposizione logica elementare* A
$$y= \begin{cases} 1 \qquad se \ A \\ 0 \qquad se \ non \ A\end{cases}$$
